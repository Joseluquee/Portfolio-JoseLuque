\# Portfolio - Jose Luque



Este repositorio reúne proyectos personales y académicos desarrollados por \*\*Jose Luque\*\*.  

Cada proyecto se encuentra en su propio notebook o carpeta y cuenta con su propia documentación y explicación.



\## Estructura



\- Cada proyecto está contenido en su propio directorio o notebook.

\- Cada proyecto incluye su \*\*README individual\*\*, que explica el objetivo, metodología y resultados.

\- Todos los notebooks están ejecutados y listos para visualizar en GitHub.



\## Tecnologías utilizadas



\- Python

\- Jupyter Notebook

\- Librerías específicas según proyecto (por ejemplo: PyTorch, NumPy, Matplotlib, etc.)



\## Cómo usar



1\. Abre el proyecto de tu interés en GitHub.

2\. Cada notebook está listo para visualizarse y ejecutarse directamente.

3\. Los datasets necesarios se descargan automáticamente en la mayoría de los notebooks.
